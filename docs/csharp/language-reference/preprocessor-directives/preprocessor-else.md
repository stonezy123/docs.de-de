# <a name="else-c-reference"></a>#else (C#-Referenz)

Mit `#else` können Sie eine zusammengesetzte bedingte Anweisung erstellen, sodass der Compiler, wenn keiner der Ausdrücke in den vorangehenden [#if](./preprocessor-if.md)-Anweisungen oder (optional) [#elif](./preprocessor-elif.md)-Anweisungen auf `true` festgelegt ist, den gesamten Code zwischen `#else` und der darauffolgenden `#endif`-Anweisung auswertet.  
  
## <a name="remarks"></a>Bemerkungen  

 [#endif](./preprocessor-endif.md) muss die nächste Präprozessoranweisung nach `#else` sein. Unter [#if](./preprocessor-if.md) finden Sie ein Beispiel zur Verwendung von `#else`.
