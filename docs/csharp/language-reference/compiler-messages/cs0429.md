# <a name="compiler-warning-level-4-cs0429"></a>Compilerwarnung (Stufe 4) CS0429

Unerreichbarer Ausdruckscode wurde entdeckt.  
  
 Dieser Fehler tritt immer dann auf, wenn ein Teil eines Ausdrucks im Code nicht erreichbar ist. Im folgenden Beispiel erf√ºllt die Bedingung `false && myTest()` dieses Kriterium, weil die Methode `myTest()` niemals ausgewertet wird, da die linke Seite der Operation `&&` immer den Wert FALSE ergibt. Sobald der Operator `&&` die Anweisung `false` als FALSE auswertet, bricht er die Auswertung ab, ohne jemals die rechte Seite auszuwerten.  
  
## <a name="example"></a>Beispiel  

 Mit dem folgenden Code wird der Fehler CS0429 generiert.  
  
```csharp  
// CS0429.cs  
public class cs0429
{  
    public static void Main()
    {  
        if (false && myTest())  // CS0429  
        // Try the following line instead:  
        // if (true && myTest())  
        {  
        }  
        else  
        {  
            int i = 0;  
            i++;  
        }  
    }  
  
    static bool myTest() { return true; }  
}  
```
